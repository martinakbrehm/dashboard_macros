# 📊 Dashboard de Aproveitamento

Dashboard interativo para monitoramento de resultados de automações de extração de dados, com foco em análise de desempenho, identificação de erros e acompanhamento operacional.

A aplicação permite visualizar métricas consolidadas a partir de arquivos Excel gerados por processos automatizados, funcionando como uma camada de observabilidade para processos de coleta e consulta de dados.

---

# 🎯 Funcionalidades

O dashboard apresenta diferentes visões para facilitar a análise dos resultados das extrações.

### 📅 Resumo diário
- Total de registros processados
- Quantidade de sucessos
- Erros de requisição
- Status ativos e inativos

### ⚠️ Distribuição de erros
- Lista das mensagens de erro mais frequentes
- Identificação de padrões de falha

### 📈 Gráfico de erros
- Visualização de erros por dia
- Visualização de erros por hora

### 🔎 Filtros dinâmicos
- Filtro por período (data)
- Filtro por empresa (quando disponível nos dados)

---

# 🧠 Arquitetura do projeto

O projeto foi estruturado em camadas para garantir melhor organização do código, separação de responsabilidades e facilidade de evolução futura.
